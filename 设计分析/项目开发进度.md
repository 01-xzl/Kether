## 🆕 2026-06-18 变更记录

### BugFix
- **TagList.vue**: 筛选后文章链接添加 `withBase()` 适配 base 路径
- **ThemeSwitch.vue**: 添加 ARIA 无障碍属性 (`aria-expanded`, `role="menu"`, `role="menuitemradio"`, `aria-checked`)

### Phase 6R giscus 迁移
- **Comments.vue**: 完全重写 — Gitalk (~245行) → giscus (~85行)，消除 clientSecret 暴露风险
- **Comments.vue**: 暗色主题通过 `transparent_dark` + `postMessage` 与网站毛玻璃风格融合
- 无其他文件改动（Layout.vue / index.ts / custom.css 保持不变）

### Phase 4 补完
- **Layout.vue**: 代码块复制按钮（`navigator.clipboard` + `MutationObserver` 路由切换适配）
- **custom.css**: 复制按钮样式（hover 显示、copied 状态）
- **index.ts**: medium-zoom 图片灯箱（路由切换重绑定）

- **custom.css**: JetBrains Mono 字体引入 (`--vp-font-family-mono`)

### Phase 7 新增

- **BgmPlayer.vue**: 新建 — 毛玻璃音乐播放器（检测音频文件 + 旋转动画 + localStorage 音量）
- **Fireworks.vue**: 新建 — Canvas 烟花粒子系统（点击触发 + 开关按钮 + 移动端降粒子数）
- **Layout.vue**: 集成 BgmPlayer + Fireworks
