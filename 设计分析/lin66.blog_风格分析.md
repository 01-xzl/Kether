| 类别 | 技术 |

## 二、视觉艺术风格

### 2.1 色彩体系

#### 亮色模式（Light — "Arona"）
| 用途 | 色值 | 说明 |
|------|------|------|
| 背景渐变 | `#b9e6f6` → `#ece5f4` | 淡蓝过渡到淡紫，清新柔和 |
| 主色/强调色 | `#128AFA` | 蓝色，用于链接和品牌元素 |
| 金色 | `#ffe401` | 标签、强调文字 |
| 文字色 | `#4c5866` | 深灰蓝，阅读友好 |
| 卡片背景 | `#ffffff` | 纯白卡片 |
| 页面背景 | `#eaeff5` | 浅灰蓝底色 |
| 三角形装饰色 | 白色，15% 不透明度 | 背景点缀，增加层次感 |

#### 暗色模式（Dark — "Plana"）
| 用途 | 色值 | 说明 |
|------|------|------|
| 背景渐变 | `#c3bde9` → `#fee4ff` | 淡紫过渡到淡粉，梦幻感 |
| 主色/强调色 | `#9d7cd8` | 紫色，替换亮色的蓝色 |
| 金色 | `#cfc6ff` | 浅紫金 |
| 文字色 | `#c8c8dc` | 浅灰紫 |
| 卡片背景 | `#1f1f2c` | 深紫黑 |
| 页面背景 | `#0f0f16` | 近乎纯黑的深色 |
| 图片亮度 | 80% | 降低图片亮度，适配暗色 |

### 2.2 整体美学特征

1. **毛玻璃（Glassmorphism）** — 导航栏、页脚、信息卡片均使用 `backdrop-filter: blur(15px)` 实现毛玻璃效果，透出底层背景纹理
2. **圆角设计** — 几乎所有元素使用大圆角（32px 最常见），营造柔和、友好的视觉感受
3. **SVG 矢量装饰** — 多处使用自定义 SVG 图形作为装饰元素：
   - 三角形阵列背景
   - 电路板/星座风格路径动画
   - 发光圆环（glow filter）
   - 抽象几何形状（用于文章卡片和代码块头部）
4. **发光效果** — 元素发光使用 `drop-shadow` + SVG filter，产生科技感 / 梦幻感
5. **二次元氛围** — 整体风格偏向 ACG（动漫游戏）文化：
   - 主题名称直接使用游戏角色名（Arona / Plana，出自《蔚蓝档案》）
   - 自定义手绘风格字体 Blueaka
   - 座右铭："邦邦卡邦 瓦尼瓦尼"（日语梗） 和 "vanitas vanitatum et omnia vanitas"（拉丁语）
   - Spine 角色播放器（类似 Live2D 看板娘）
6. **纹理与图案** — 背景使用重复线性渐变生成三角形纹理，代码块和文章卡片头部有装饰性 SVG 图案

### 2.3 排版与字体

| 用途 | 字体 | 说明 |
|------|------|------|
| 正文/标题 | **Blueaka**（自定义） | 手绘风格中文书法字体，圆润可爱，辨识度高 |
| 加粗标题 | **Blueaka_Bold** | 同上加粗版本 |
| 代码 | **JetBrains Mono** | 现代等宽字体，支持多种 Unicode 范围 |
| 图标 | **iconfont**（自定义） | 社交图标（GitHub、Bilibili、微博等）、UI 图标（搜索、标签、箭头、时间） |

- 字体引入方式：通过 @font-face 加载 woff2/woff 格式
- 代码块设计：圆角、顶部有装饰条和语言标签、复制按钮

### 3.2 响应式设计

- **桌面端**（>768px）：完整布局，导航菜单水平展开
- **移动端**（≤768px）：
  - 导航栏高度降至 64px
  - 汉堡菜单代替水平导航
  - 文章卡片改为垂直排列，封面图变宽
  - 字体和间距等比例缩小
  - 下拉菜单改为更紧凑的布局
  - 文章标题字号从 4.5vw 降为 5vh

### 3.3 动画与交互

| 元素 | 动画效果 |
|------|----------|
| Splash 页面 | SVG 发光路径缓慢呼吸动画 |
| Banner | 加载时模糊→清晰（fade-blur-in），背景位置固定视差 |
| 波浪 | Canvas 实现的实时波浪动画 |
| 导航链接 | 悬浮平移 + 颜色变化 |
| 文章卡片 | 悬浮阴影变化，hover 效果 |
| 分页箭头 | 无限循环左右摇摆动画 |
| 回到顶部 | 平滑渐入渐出 + 位置动画 |
| 烟花 | 用户操作时触发 Canvas 烟花粒子 |
| 头像 | 悬浮时 360° 旋转 + 放大 |
| 主题切换 | 无闪烁切换（使用 localStorage 预检测） |
| 搜索弹窗 | 淡入 + 缩放弹出动画 |

## 四、特色功能

### 4.1 主题切换系统
- 三种模式：亮色（Arona）、暗色（Plana）、跟随系统（System）
- 切换时所有 CSS 变量联动更新：颜色、纹理、背景图、发光色、SVG 图标颜色
- 自动 detect 系统偏好（`prefers-color-scheme`）
- 支持 localStorage 持久化

### 4.2 Spine 角色
- 右下角可交互的 2D 骨骼动画角色
- 支持播放/暂停、速度控制、皮肤切换、动画切换
- 角色对话气泡功能
- 可全局开关启用/禁用

### 4.3 背景音乐
- 自动循环播放（需用户交互后才播放）
- 通过 audio 标签 + JS 控制
- 音乐控制按钮在菜单中

### 4.4 搜索功能
- 全站文章搜索
- 弹出式搜索对话框，带毛玻璃背景
- 输入框聚焦时高亮边框

### 4.5 评论系统
- Gitalk 集成，使用 GitHub OAuth
- 与暗色/亮色主题同步样式

## 五、设计亮点总结

1. **沉浸式开屏** — SVG 动画加载页给人第一印象深刻，几何抽象图形 + 发光效果充满科技感
2. **统一的设计语言** — 圆角、毛玻璃、发光效果贯穿全站，视觉一致性极高
3. **双主题独立设计** — 不是简单的颜色反转，而是两套完整的视觉方案（蓝↔紫、清新↔梦幻）
4. **二次元文化融合** — 从字体选择到主题命名到角色系统，完整营造 ACG 氛围，目标用户明确
5. **细腻的动效** — 每个交互都有对应的微动效（hover、transition、keyframe 动画），提升使用愉悦感
6. **个性装饰丰富** — SVG 装饰、背景纹理、发光路径等元素让页面不单调，有手工艺感
7. **功能与美学平衡** — 在保持视觉特色的同时，评论、搜索、标签分类等功能一应俱全

## 六、可借鉴的设计思路（供复刻参考）

1. **SVG 品牌标识** — 使用自定义 SVG 作为 Logo，风格统一且可缩放
2. **CSS 变量驱动的主题** — 通过 `:root` 和 `html[theme=dark]` 的 CSS 变量实现一键切换
3. **文章卡片装饰逻辑** — 左侧色条 + 圆点指示器 + 装饰背景图，简单但有层次
4. **代码块美化** — 顶部装饰条 + 语言标签 + 复制按钮，提升技术博客的专业感
5. **毛玻璃导航与页脚** — 导航栏固定在顶部，与页面内容重叠时透出背景，视觉高级
6. **个性化角落** — 座右铭循环打字效果、角色对话等，增加趣味性和辨识度
7. **色彩的心理暗示** — 亮色用蓝（冷静/技术）→暗色用紫（神秘/梦幻），情绪过渡自然
