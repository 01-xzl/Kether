# Godot 学习路径（精简版）

对标横板跳跃游戏开发的快速上手教程。

---

## 学习目标

- 掌握 Godot 编辑器基础操作
- 理解 `Node`、`Scene`、`Signal` 三个核心概念
- 学会 `GDScript` 基础语法
- 能制作一个可控制的玩家角色和基础平台

---

## 核心学习路线

### 第 1 阶段：编辑器认识（1 天）

**关键区域：**
- `Scene`：节点树
- `Inspector`：属性编辑
- `FileSystem`：资源管理
- `2D / Script / Game`：工作区

**实操：**
- 新建场景，添加 `Label` 和 `Button`
- 在 `Inspector` 修改文字和位置
- 创建 `scenes/`, `scripts/`, `assets/` 文件夹

**完成标准：** 能新建场景，添加节点，修改属性

---

### 第 2 阶段：三个核心概念（2 天）

#### Node（节点）
- 最基础的构成单位
- 可以组成父子关系
- 记住：一切对象都是节点

**练习：** 新建 `Node2D` 场景，添加 `Sprite2D` 和 `Label`，调整位置

#### Scene（场景）
- 节点的打包组合
- 可以复用和实例化
- 一个完整的游戏单元

**练习：** 做一个角色场景 `Player.tscn`，再在主场景中实例化它

#### Signal（信号）
- 事件通知机制
- 按钮点击、碰撞进入、动画结束都是信号

**练习：** 连接 `Button` 的点击信号，点击时改变 `Label` 文字

---

### 第 3 阶段：GDScript 基础（3 天）

**必学内容：**
```
var x = 10              # 变量
const MAX_SPEED = 100   # 常量
func move():            # 函数
    if x > 0:          # 条件
        print("移动")
```

**生命周期函数：**
- `_ready()`：进入场景时执行一次
- `_process(delta)`：每帧执行
- `_input(event)`：处理输入

**练习：**
1. 定义角色生命值变量
2. 写一个受伤函数
3. 用条件判断生命值是否为 0

---

### 第 4 阶段：2D 核心功能（4 天）

#### 输入系统
```
if Input.is_action_pressed("ui_right"):
    position.x += speed
```

#### 碰撞系统
- `CharacterBody2D`：可移动角色
- `CollisionShape2D`：碰撞形状
- `Area2D`：检测区域

#### 动画系统
- `AnimationPlayer`：播放动画
- 属性动画：改变节点属性

#### UI 基础
- `Label`：显示文字
- `Button`：交互按钮
- `Container`：布局容器

**练习：** 制作一个角色可上下左右移动，且无法穿过障碍物

---

### 第 5 阶段：三个小项目（3 天）

**项目 1：按钮交互**
- 点击按钮改变标签文字

**项目 2：角色移动**
- 四方向移动，碰撞检测

**项目 3：角色信息面板**
- 显示 HP、攻击力等数据

---

## 针对横板跳跃游戏的优先学习

前 4 个阶段后，重点深入这些模块：

### 重力与物理（5 天）
```
velocity.y += gravity * delta
velocity = move_and_slide(velocity)
```

### 跳跃机制（3 天）
- 检测是否在地面上
- 按跳跃键施加向上速度
- 可选：双跳、墙跳

### 敌人与碰撞反应（4 天）
- 敌人简单移动
- 触碰敌人游戏结束或扣血
- 踩敌人击败敌人（可选）

### 平台与关卡（3 天）
- 放置平台
- 可移动平台
- 关卡切换

---

## 总建议

- **不要一次学完所有**，按顺序来
- **每学一个知识点，马上做小练习**
- **先做出最小可运行结果，再扩展功能**
- **遇到问题先试着自己改，再查文档**

推荐总耗时：**2-3 周**

---

## 快速参考

| 要素 | 节点类型 | 用途 |
|-----|--------|------|
| 角色 | `CharacterBody2D` | 可移动可碰撞 |
| 敌人 | `CharacterBody2D` / `Area2D` | 移动 + 检测 |
| 平台 | `StaticBody2D` / `CollisionShape2D` | 静止碰撞体 |
| 检测区域 | `Area2D` + 信号 | 进入/退出检测 |
| UI | `Control` 系列 | 界面和菜单 |
| 动画 | `AnimationPlayer` | 角色动画播放 |

---

## 常见快捷操作

- `Ctrl+N`：新建脚本
- `F5`：运行当前场景
- `F6`：运行项目
- `Ctrl+S`：保存
